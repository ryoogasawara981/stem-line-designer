トークンの作り方(初回だけ)
  1. こちらを開く(GitHubにログイン済みで)
  2. Repository access →「Only select repositories」→ stem-line-designer を選択
  3. Permissions →「Repository permissions」→ Contents を Read and write に、Actions を Read and write に設定
  4. 「Generate token」を押して表示された文字列をコピー
  5. 上の欄に貼り付けて「保存する」。この端末のブラウザに保存され、以後入力不要
